# Image Converter

## Overview
The **Image Converter** application is an app that allows you to convert images and PDFs between various formats

- **Image to Image :**
  - JPG/JPEG to PNG
  - PNG to JPG/JPEG

- **Image to PDF :**
  - JPG/JPEG to PDF
  - PNG to PDF

- **PDF to Image :**
  - PDF to JPG/JPEG
  - PDF to PNG

## Requirements
- ** Operating System : **
  - Mac OS
  - Windows 10/11
  - Linux
- **Java Version :** 
  - Minimum : Java 11 LTS
  - Recommended : Java 21 LTS

 ## Usage
 - Double click the image-converter.jar
 - Or go to command-line and type java -jar image-converter.jar (if double-clik does nothing)
